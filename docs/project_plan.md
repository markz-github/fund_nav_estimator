# 开发计划

## 项目目标

构建一个基金当日净值估算系统，包含：

- 维护自选基金列表。
- 自动获取基金官方净值。
- 自动维护基金持仓。
- 获取底层资产当日行情。
- 每半小时估算自选基金的当日净值。
- 在前端页面展示官方净值、估算净值、更新时间、持仓和异常状态。

## 5. 当日估算净值计算

目标：根据基金最新官方净值、持仓比例和当日行情，计算基金当日估算净值。

### 计算公式

```text
估算涨跌幅 = Σ(底层资产当日涨跌幅 × 持仓比例)
当日估算净值 = 最新官方净值 × (1 + 估算涨跌幅)
```

### 已完成

- [已完成] 获取基金最新官方净值。
- [已完成] 获取基金最新报告期持仓。
- [已完成] 获取每个持仓资产最新行情。
- [已完成] 计算加权涨跌幅。
- [已完成] 计算估算净值。
- [已完成] 计算有效持仓覆盖比例。
- [已完成] 写入 `fund_estimates`。
- [已完成] 对缺失净值、缺失持仓、缺失行情进行跳过处理并返回原因。

### 待处理

- [已完成] 对缺失净值、缺失持仓、缺失行情写入异常记录。
- [已完成] 前端展示估算运行结果和跳过原因。

### 覆盖比例示例

```text
基金披露持仓总比例 = 85%
其中成功获取行情的持仓比例 = 72%
coverage_ratio = 72% / 85%
```
